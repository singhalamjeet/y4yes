# y4yes.com - Network Tools Suite

A comprehensive suite of network tools built with Next.js and Tailwind CSS.

## Features

- **What's My IP**: Public IP detection.
- **DNS Lookup**: Resolve A, MX, TXT records.
- **WHOIS Lookup**: Domain registration details.
- **SSL Checker**: Certificate validity and expiration.
- **Port Scanner**: Check open ports.
- **Ping Test**: Measure latency.
- **Traceroute**: Map network paths.
- **Speed Test**: Latency and bandwidth simulation.
- **URL Tools**: Encoder/Decoder.

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tech Stack

- **Framework**: Next.js 15
- **Styling**: Tailwind CSS v4
- **Language**: TypeScript
